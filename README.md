# Modified Connect Four

## Compilation

```bash
javac ./**/*.java
```

## Usage

> [!IMPORTANT]
> The working directory for the game is the `src/main/java` folder; therefore, you must run the following command inside of the `src/main/java` folder:
>
> ```bash
> cd ./src/main/java/
> ```

```bash
java cielsachen.ccdstru.Main
```
